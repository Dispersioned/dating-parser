# VK Better Dating client

## Установка
1. Установить nodejs 18 версии
2. Открыть папку проекта в любом терминале
3. Прописать `npm i -g yarn`
4. Прописать `yarn`
5. Прописать `yarn install-deps`
6. Запустить проект командой `yarn start`
